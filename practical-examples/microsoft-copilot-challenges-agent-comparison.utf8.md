Lecture transversale

Security Copilot pour incidents secu; observability agent pour SRE
Run

Purview/Fabric/Defender si disponible; GitHub pour controles code
Secure

Foundry pour agent pro-code; Copilot Studio pour low-code
Build

Azure Migrate pour inventaire; GitHub pour backlog/code
Move

Copilot for Azure/FinOps pour analyse; GitHub pour automatiser
Optimize
Choisir l'agent specialise quand il sait deja faire; utiliser GitHub Copilot quand il faut coder, connecter ou industrialiser.
